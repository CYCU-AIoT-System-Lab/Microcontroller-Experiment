u32  AES_SR_x

Check whether the specified AES status has been set.

Parameters
HT_AESnwhere HT_AESn is the selected AES peripheral.
AES_SR_xspecify the flag to be check. This parameter can be one of the following values:
  • AES_SR_IFEMPTY : AES Input FIFO is Empty
  • AES_SR_IFNFULL : AES Input FIFO is not Full
  • AES_SR_OFNEMPTY : AES Output FIFO is not Empty
  • AES_SR_OFFULL : AES Output FIFO is Full
  • AES_SR_BUSY : AES is busy when AES is in encrypt/decrypt action and key expansion
Return values
SETor RESET
