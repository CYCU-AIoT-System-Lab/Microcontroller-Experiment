◆ SLED_FIFOTrigLevelConfig()

void SLED_FIFOTrigLevelConfig ( HT_SLED_TypeDef *  SLEDx,
u8  FifoLevel 
)

Set the specified SLED FIFO trigger level.

Parameters
SLEDxwhere SLEDx is the selected SLED from the SLED peripherals.
FifoLevelspecify the FIFO trigger level. This parameter can be one of the following values:
  • SLED_FIFO_LEVEL_0 : data request will be inserted when FIFO data count is equal to 0
  • SLED_FIFO_LEVEL_1 : data request will be inserted when FIFO data count is less than or equal to 1
  • SLED_FIFO_LEVEL_2 : data request will be inserted when FIFO data count is less than or equal to 2
  • SLED_FIFO_LEVEL_3 : data request will be inserted when FIFO data count is less than or equal to 3
Return values
None
