◆ LEDC_Init()

void LEDC_Init ( LEDC_InitTypeDef *  LEDC_InitStruct)

Initialize the LEDC peripheral according to the specified parameters in the LEDC_InitStruct.

Parameters
LEDC_InitStructPointer to a LEDC_InitTypeDef structure. Please note the following.
  1. When LEDC is started, the LEDC_ClockSource, LEDC_ClockPrescaler and LEDC_DeadTime can't be changed, so LEDC_Init() will turn off the LED first.You need to restart LEDC by LEDC_Cmd(ENABEL).
  2. The LEDC_DeadTime number must be less than the LEDC_Prescaler. Example: If LEDC_DutyClockNumber selects LEDC_DTYNUM_8, the valid LEDC_DeadTime ranges from 0 to 7. If LEDC_DutyClockNumber selects LEDC_DTYNUM_16, the valid LEDC_DeadTime ranges from 0 to 15. If LEDC_DutyClockNumber selects LEDC_DTYNUM_32, the valid LEDC_DeadTime ranges from 0 to 31. If LEDC_DutyClockNumber selects LEDC_DTYNUM_64, the valid LEDC_DeadTime ranges from 0 to 63.
Return values
None
