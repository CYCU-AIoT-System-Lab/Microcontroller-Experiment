void GPIO_LockPinTest (void)
 Lock Pin Test. (Port A Pin 1) PA1 is Locked, so changing PA1 GPIO Configuration is invalid. After changing GPIO Configuration, PA1 GPIO Configuration as below: AFIO Mode: GPIO Mode Default Value : RESET Pull Resistor : Pull-DISABLE Output Driving Current: 8mA Direction : Output Result => PA1 Toggle Ten Times, because PA1 is Locked (Change Configuration is invalid) PA1: (Output -> Output) Mode, (Pull-Disable -> Pull-Disable)
 
void GPIO_UnLockPinTest (void)
 UnLock Pin Test. (Port A Pin 2) PA2 is UnLocked, so changing PA2 GPIO Configuration is valid. After changing GPIO Configuration, PA2 GPIO Configuration as below: AFIO Mode : GPIO Mode Default Value : RESET Pull Resistor : "Pull-Up" Output Driving Current: 8mA Direction : "Input" Result => PA2 keeps the high level, because PA2 is unlocked (Change Configuration is valid) PA2: (Output -> Input) Mode, (Pull-Disable -> Pull-Up)
