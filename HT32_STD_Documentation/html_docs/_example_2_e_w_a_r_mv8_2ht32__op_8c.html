#define WDT_OPT   (0)

Watchdog Timer Auto Enable: 1: Enable, 0: Disable
WDT will be enabled immediately when the MCU
power on reset or system reset occurs.
The WDT then can be disabled by firmware.
The default timeout period of WDT is 8 second. (32 kHz LSI, Prescaler = 1/64, 12-bit counter)
