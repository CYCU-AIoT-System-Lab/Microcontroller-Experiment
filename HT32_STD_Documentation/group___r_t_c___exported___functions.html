void RTC_OutConfig ( RTC_ROWM_Enum  WMode,
RTC_ROES_Enum  EventSel,
RTC_ROAP_Enum  Pol 
)

Configure the RTC output function.

Parameters
WModespecify the RTC output waveform mode This parameter can be one of the following values:
  • RTC_ROWM_PULSE : Pulse mode
  • RTC_ROWM_LEVEL : Level mode
EventSelspecify the RTC output event selection This parameter can be one of the following values:
  • RTC_ROES_MATCH : Compare match selected
  • RTC_ROES_SECOND : Second clock selected
Polspecify the RTC output active polarity This parameter can be one of the following values:
  • RTC_ROAP_HIGH : Active level is high
  • RTC_ROAP_LOW : Active level is low
Note
This function will disable RTC output first.
